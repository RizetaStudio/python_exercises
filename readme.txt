
======================================
      Ejercicios Prácticos en Python
======================================

Este repositorio contiene una colección de ejercicios prácticos realizados durante el curso de Python en Mastermind. Cada ejercicio refleja diferentes conceptos y habilidades aprendidas, desde estructuras básicas hasta el uso de funciones y lógica aplicada.

Contenido del repositorio:
--------------------------

1. **Combate Pokémon**
   - Descripción: Simula un combate por turnos entre Pikachu y Squirtle con barra de vida dinámica y ataques únicos.
   - Conceptos aplicados: Uso de ciclos, lógica condicional y manipulación de variables.

2. **Conversor de Monedas**
   - Descripción: Convierte entre diferentes monedas (euros, dólares, libras).
   - Conceptos aplicados: Input del usuario, operaciones matemáticas y estructuras condicionales.

3. **Descuento en el Cine**
   - Descripción: Determina si el usuario cumple los requisitos para obtener un descuento en su entrada al cine.
   - Conceptos aplicados: Operadores lógicos y condicionales.

4. **Elección de Teléfono**
   - Descripción: Un test interactivo para recomendar el mejor móvil según tus preferencias.
   - Conceptos aplicados: Inputs anidados y salida dinámica basada en opciones.

5. **Ejercicios con 'for'**
   - Descripción: Incluye ejercicios como contar vocales, generar tablas de multiplicar, y contar espacios, puntos y comas en una frase.
   - Conceptos aplicados: Uso del ciclo 'for', iteración y lógica básica.

6. **Lista de la Compra**
   - Descripción: Permite crear y gestionar una lista de la compra interactiva.
   - Conceptos aplicados: Ciclos 'while', listas y validación de entradas.


Cómo ejecutar los ejercicios:
-----------------------------
1. Clona el repositorio:
   git clone <url_del_repositorio>

2. Navega al directorio donde guardaste el repositorio.

3. Abre el archivo del ejercicio que quieras probar en tu IDE de preferencia o en la terminal:
   python nombre_del_archivo.py
